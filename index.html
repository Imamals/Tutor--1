01. The Basics

Variables & Types

Think of variables as containers for storing data. In Python, you don't need to declare types explicitly.

x = 10
msg = "I love Python"
is_fun = True

Lists

Lists are used to store multiple items in a single variable. They are ordered and changeable. 

fruits = ["apple", "banana"]
fruits.append("orange")
print(fruits)

02. User Input

The input() Function

To interact with the user, we use the input() function. It pauses the program and waits for the user to type something.

name = input("What is your name? ")
print(f"Hello, {name}!")

02. Control Flow

Making Decisions

Python uses if, elif, and else for logical conditions.

age = 18
if age >= 18:
    print("You are an adult")
else:
    print("You are a minor")

03. Functions

Reusable Blocks

A function is a block of code which only runs when it is called.

def greet(name):
    return f"Hello, {name}!"

print(greet("Alice"))
