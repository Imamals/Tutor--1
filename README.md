🏆 Sensei.py
Python Logic. Mastered.
Sensei.py is a minimalist, interactive coding platform built to prioritize conceptual mastery over rigid syntax. By leveraging a custom AST (Abstract Syntax Tree) engine, the platform evaluates the underlying structure of your code rather than performing strict string matching. It’s designed for learners who want to master the "how" of programming without being punished for a typo.
✨ Features
Conceptual Evaluation: Our backend uses Python's ast module to verify your logic (Assignments, Loops, Functions), ignoring variable names or exact formatting.
Minimalist UI: A high-voltage, dark-mode interface with subtle glassmorphism, designed for deep technical focus.
Instant Validation: Receive immediate feedback on your logic the moment you run your code.
Logic-First Philosophy: We believe that if the conceptual "First Touch" is correct, the syntax will naturally follow.
🛠️ Tech Stack
Backend: Python 3 (Flask).
Logic Engine: Abstract Syntax Trees (AST).
Frontend: HTML5, CSS3 (Flexbox/Grid), Vanilla JavaScript.
🧠 The Philosophy
"It’s not about the semicolons you miss; it’s about the logic you find."
Most platforms act like rigid unit tests. Sensei.py acts like a focused mentor, recognizing the validity of an elegant solution even if it's not textbook perfect.
